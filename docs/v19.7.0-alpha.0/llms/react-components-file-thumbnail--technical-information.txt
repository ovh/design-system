## FileThumbnail

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| disabled | `boolean` | - | `undefined` | Whether the component is disabled. |
| dismissible | `boolean` | - | `true` | Whether the component holds a dismiss button. |
| error | `string` | - | `undefined` | The file error message to display. |
| file | `File` |  | `undefined` | The current File object. |
| i18n | `Partial` | - | `undefined` | Internal translations override. |
| locale | `LOCALE` | - | `undefined` | The locale used for size formatting and the translation of the internal elements. |
| onFileRemove | `(detail: FileThumbnailRemoveDetail) => void` | - | `undefined` | Callback fired when the file is removed. |
| progress | `number` | - | `undefined` | The file upload progress. |
