## Description
The cart-manager is a smart component with all the logical stuff about items management.

It uses several components :
- `cart`: the main component to use to present the cart
- `cart-section`: a section container that contains items and options
- `cart-item`: a line for an item with a price
- `cart-item-option`: a line for an item's option with a price
- `cart-footer`: display the total amount

## Tests

**header**
- should displays the passed content in the header zone

**content**
- should displays the main content in the main content zone

**footer**
- should displays the passed content in the footer zone

**size**
- should have `sm` size by default

**empty**
- should display no items if empty set to true

**method:getTotalAmount**
- should returns the total amount excluded VAT by default
- should returns the total amount with VAT if specified

**method:getItemQuantity**
- should returns the number of item placed in the cart
