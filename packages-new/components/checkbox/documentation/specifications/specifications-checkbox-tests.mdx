### Controller

- **spec:ods-checkbox-controller**
  - should initialize
  - methods
    - afterInit
      - should call propagateCheckedToChild and propagateDisabledToChild methods
      - should disable focus management on child
      - should warn user if no checkboxable element found
    - getTabIndex
      - should get the tab index
      - should get no tab index if disabled
    - handleToggleByClick
      - should call toggleCheck method on click
    - handleToggleByKeyEvent
      - should call toggleCheck method on key event
    - onBlur
      - should update blur state
      - should do nothing if disabled
    - onFocus
      - should call setFocus
    - propagateCheckedToChild
      - should propagate checked to child
    - propagateDisabledToChild
      - should propagate disabled to child
    - propagateHasFocusToChild
      - should propagate hasFocus to child
    - setFocus
      - should do nothing if disabled
      - should do nothing if no inputEl
      - should update focus state
    - setTabindex
      - should set the index
    - toggleCheck
      - should not toggle if disabled and not checked
      - should not toggle if disabled and checked
    - optimistic update
      - should doing optimistic update
    - pessimistic update
      - saving process in success
        - should call beforeSave
        - should call save
        - should call afterSave
      - saving catching potential errors
        - should catch potential error of beforeSave
        - should catch potential error of save
        - should catch potential error of afterSave
