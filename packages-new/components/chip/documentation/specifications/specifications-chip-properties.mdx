| Name             | Type                   | default     | Description                          |
|-- -------------- | ---------------------- | ----------- | ------------------------------------ |
| `[color]`        | `string`               | `default`   | main color                           |
| `[contrasted]`   | `boolean`              | `false`     | contrasted or not                    |
| `[disabled]`     | `boolean`              | `false`     | disabled state                       |
| `[flex]`         | `boolean`              | `false`     | full width or not                    |
| `[size]`         | **`ChipSize`**         | `md`        | size                                 |
| `[variant]`      | **`ChipVariant`**      | `flat`      | variant form                         |
| `[removable]`    | `boolean`              | `false`     | is the chip is removable             |
| `[selectable]`   | `boolean`              | `false`     | is the chip can be selected or not   |
