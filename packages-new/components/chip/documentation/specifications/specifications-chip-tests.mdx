**Contrast**
    - should have contrasted chip if contrasted
    - should not be contrasted by default

**Size**
    - should have size chip to `md` by default

**Disabled**
    - should not be disabled by default
    - should be not clickable if disabled
    - should not emit event if disabled

**Basic variant**
    - should have no background
    - should set color on text content

**Flat variant**
    - should set color on background
    - should set contrasted color on text content

**Stroked variant**
    - should set color on border
    - should set color on text content
