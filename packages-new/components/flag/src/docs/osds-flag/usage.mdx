## Usage

You have to setup ods and call Ods.instance().assetPath() before using Flag component.
For more details about assets, see [Installation](?path=/docs/code-stencil-components-how-to-install--page#assets) page.

You can have an example of usage in our three Starters for:
[React](?path=/story/code-starters-react-react--page),
[React Vite](?path=/story/code-starters-react-react-vite--page),
[Vue Vite](?path=/story/code-starters-vue-vue-vite--page)

### Default
will set `fr` flag by default
<div class="container">
  <osds-flag></osds-flag>
</div>

```html
<osds-flag></osds-flag>
```

### Basic
<div class="container">
  <osds-flag iso='de'></osds-flag>
</div>

```html
<osds-flag iso='de'></osds-flag>
```

### Custom asset path
For some cases, you may want to use a custom path that will override the ODS path:
<div class="container">
  <osds-flag iso="de" asset-path="../flags-custom-path/"></osds-flag>
</div>

```html
<osds-flag iso="de" asset-path="../flags-custom-path/"></osds-flag>
```

### Custom src file
Same goes with `src` attribute when you want to point at your own SVG:
<div class="container">
  <osds-flag iso="de" src="../flags-custom-path/it.svg"></osds-flag>
</div>

```html
  <osds-flag iso="de" src="../flags-custom-path/it.svg"></osds-flag>
```

### Lazy loading
will load the flag only when it is visible in viewport
<div class="container">
  <osds-flag iso="gb" lazy></osds-flag>
</div>

```html
  <osds-flag iso="gb" lazy></osds-flag>
```
