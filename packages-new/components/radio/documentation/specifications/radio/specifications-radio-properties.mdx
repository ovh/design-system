| Name         | Type      | default | Description                 |
|--------------|-----------|---------|-----------------------------|
| `[ariaLabel]`     | `string`  | `''`    | Radio aria label                  |
| `[ariaLabelledby]`     | `string`  | `''`    | The id to an external description                  |
| `[beforeSave]`     | `({ checked, value }: { checked: boolean, value?: string }) => Promise<void>`  | `-`    | before save input allow to set a function that returns a promise. It is called before each time a save is performed                  |
| `[save]`     | `({ checked, value }: { checked: boolean, value?: string }) => Promise<void>`  | `-`    | save input allow to set a function that returns a promise. It is called before each time an update is performed and allowing to manage pessimistic update strategy                  |
| `[afterSave]`     | `({ checked, value }: { checked: boolean, value?: string }) => Promise<void>`  | `-`    | after save input allow to set a function that returns a promise. It is called after each time a save is performed                  |
| `[checked]`     | `boolean`  | `false`    | The checked status of the radio                  |
| `[checking]`     | `boolean`  | `false`    | If the radio is under checking state (asynchronous)                  |
| `[disabled]` | `boolean` | `false` | If radio is disabled or not |
| `[label]`     | `string`  | `-`    | Radio label                  |
| `[name]`     | `string`  | `-`    | Radio input name                  |
| `[value]`    | `any`     | `-`    | Radio value                 |
