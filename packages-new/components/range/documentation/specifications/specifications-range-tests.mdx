**color**
    - should be a `string`
    - should have color to `default` by default
**Disabled**
    - should be a `boolean`
    - should not be disabled by default
    - should be not clickable if disabled
    - should not emit event if disabled
**flex**
    - should be a `number` or `void`
    - should have size to `100%` if `void`
