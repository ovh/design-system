### Accessibility

The usage of a `<label>` tag for the label slots adds a native browser behaviour:
On click on the label, the browser sets the focus on any native textarea linked using the `for` attribute.

<Canvas withSource="none">
  <label for="midable">
    <osds-text level='heading' color="primary">
      An accessible label
    </osds-text>
  </label>
  <osds-textarea id="midable"></osds-textarea>
</Canvas >

```html
<label for="midable">
  <osds-text level='heading' color="primary">
    An accessible label

</label>
```
