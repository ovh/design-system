## Description
The cart is a presentational component and displays the contents of a cart.
It is made of several subcomponents :
- `cart`: the main component to use
- `cart-section`: a section container that contains items and options
- `cart-item`: a line for an item with a price
- `cart-item-option`: a line for an item's option with a price
- `cart-footer`: display the cart footer item and the cart total
- `cart-footer-item`: a line for an item with a price and optional extra
- `cart-total`: display the total price and total quantity of product for the cart

The cart manager is the smart version of the cart with computing.

## Contents

| Name        | Description                                                    |
|-------------|----------------------------------------------------------------|
| `header` | header content like a title                                       |
| `content` | main content displayed. Set here the cart-section                |
| `footer` | footer content to display. You can put some button inside it here |
| `cart-empty` | displayed content when there are no items in the cart         |
