#### Basic use

*with `tile` component*

```html
<!-- stencil example basic -->
<osds-checkbox value="my-value" name="my-input-name">
  <osds-tile interactive>Radio with Tile in it</osds-tile>
</osds-checkbox>

<!-- stencil example checked -->
<osds-checkbox value="my-value" name="my-input-name" checked>
  <osds-tile interactive>Radio with Tile in it</osds-tile>
</osds-checkbox>

<!-- stencil example disabled -->
<osds-checkbox value="my-value" name="my-input-name" disabled>
  <osds-tile interactive>Radio with Tile in it</osds-tile>
</osds-checkbox>
```

#### Standalone use

*use with your own `vanilla` or your component*

```html
<!-- stencil example -->
<style>
  .check-button:not([checked]) {
    background: #8f562d;
  }

  .check-button[checked] {
    background: green;
  }

  .check-button[has-focus] {
    border: 2px solid black;
  }
</style>
<osds-checkbox value="my-value" name="my-input-name">
  <div class="check-button">your DOM element to be decorated as checkbox</div>
</osds-checkbox>
```
