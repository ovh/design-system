## Description
The cart-manager is a smart component with all the logical stuff about items management.

It uses several components :
- `cart`: the main component to use to present the cart
- `cart-section`: a section container that contains items and options
- `cart-item`: a line for an item with a price
- `cart-item-option`: a line for an item's option with a price
- `cart-footer`: display the total amount
