The nominal way to use it is with another component that will manage all the visual aspect only.

#### With Tile component

<Canvas withSource="none">
  <osds-checkbox value="my-value" name="my-input-name">
    <osds-tile hoverable>Radio with Tile in it</osds-tile>
  </osds-checkbox>
</Canvas >

```html
<osds-checkbox value="my-value" name="my-input-name">
  <osds-tile hoverable>Radio with Tile in it</osds-tile>
</osds-checkbox>
```

> Here the Tile with hoverable attribute to have the styles of interactions (focus, hover, active and so on).
