## Usage

You have to setup ods and call Ods.instance().assetPath() before using Flag component.

For more details about assets, see [Get Started - Assets](?path=/story/ovhcloud-design-system-get-started--page#assets) page.

### Default
will set `fr` flag by default
<div style={{width: '100px'}}>
  <osds-flag></osds-flag>
</div>

```html
<osds-flag></osds-flag>
```

### Basic
<div style={{width: '100px'}}>
  <osds-flag iso='de'></osds-flag>
</div>

```html
<osds-flag iso='de'></osds-flag>
```

### Custom asset path
For some cases, you may want to use a custom path that will override the ODS path:
<div style={{width: '100px'}}>
  <osds-flag iso="de" asset-path="./custom-flags/"></osds-flag>
</div>

```html
<osds-flag iso="de" asset-path="./custom-flags/"></osds-flag>
```

### Custom src file
Same goes with `src` attribute when you want to point at your own SVG:
<div style={{width: '100px'}}>
  <osds-flag iso="de" src="./custom-flags/it.svg"></osds-flag>
</div>

```html
  <osds-flag iso="de" src="./custom-flags/it.svg"></osds-flag>
```

### Lazy loading
will load the flag only when it is visible in viewport
<div style={{width: '100px'}}>
  <osds-flag iso="gb" lazy></osds-flag>
</div>

```html
  <osds-flag iso="gb" lazy></osds-flag>
```
