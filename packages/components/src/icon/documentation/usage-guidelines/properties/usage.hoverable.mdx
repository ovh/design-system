Use `hoverable` attribute so Icon color can change when in selectable parent.

With Link component as selectable parent:

<osds-link color='primary'>
  Link with hoverable Icon
  <span slot='start'><osds-icon name="arrow-left" size="xs" hoverable></osds-icon></span>
</osds-link>
