### Accessibility

Tabulation can be used to navigate through focusable elements added to rows.

- Table header `<th>`
- `<th scope="col">`
- `<th scope="row">`
- Add a `<caption>` inside the `<table>`

More on [WAI ARIA documentation from W3C for tables](https://www.w3.org/WAI/tutorials/tables/)

*Large content on small spaces*

Keep the Table headers visible while scrolling by setting `<th>` elements to a sticky position.

```scss
table {
  position: relative;
}

thead {
  tr {
    th {
      background-color: #fff;
      position: sticky;
      top: 0;
    }
  }
}
```

Keep the first column sticky while scrolling:

```scss
table {
  position: relative;
}

tbody {
  tr {
    th,
    td {
      background-color: #fff;

      &:first-child {
        left: 0;
        position: sticky;
      }
    }
  }
}
```

Or keep both sticky:

```scss
table {
  position: relative;
}

thead {
  tr {
    th {
      background-color: #fff;
      position: sticky;
      top: 0;
      z-index: 2;

      &:first-child {
        left: 0;
        z-index: 3;
      }
    }
  }
}

tbody {
  tr {
    th,
    td {
      background-color: #fff;

      &:first-child {
        left: 0;
        position: sticky;
      }
    }
  }
}
```

More recommendations on [MDN documentation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/table#displaying_large_tables_in_small_spaces).
