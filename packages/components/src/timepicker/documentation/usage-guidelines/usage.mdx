## Usage

### Default
<osds-timepicker></osds-timepicker>

```html
<osds-timepicker></osds-timepicker>
```

### Example with all timezones
<osds-timepicker timezones='all'></osds-timepicker>

```html
<osds-timepicker timezones='all'></osds-timepicker>
```

### With specific timezones
<osds-timepicker timezones='["utc+0", "utc+1", "utc+2"]'></osds-timepicker>

```html
<osds-timepicker timezones='["utc+0", "utc+1", "utc+2"]'></osds-timepicker>

```

### Example with all timezones and in error
<osds-timepicker timezones='all' error=''></osds-timepicker>

```html
<osds-timepicker timezones='all' error=''></osds-timepicker>
```

### Example with all timezones and disabled
<osds-timepicker timezones='all' disabled=''></osds-timepicker>

```html
<osds-timepicker timezones='all' disabled=''></osds-timepicker>
```
