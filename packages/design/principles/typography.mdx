# Typography

## Font-family
Source Sans Pro is the main font used in the OVHcloud universe. It can be used for headings as well as paragraphs.

The related fallback fonts are :
`"Source Sans Pro", "Trebuchet MS", Arial, "Segoe UI", sans-serif`
