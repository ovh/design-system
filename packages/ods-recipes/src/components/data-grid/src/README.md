# Data Grid

This recipe builds on the Data Table component to provide a complete, interactive data grid experience.
It comes with a toolbar that includes search, action buttons, as well as pagination controls and per-row action menus.

Use this recipe when you need to present structured data that users can explore and interact with efficiently.
It is particularly suited for dashboards, admin panels, or any interface requiring data management.

Use Cases:
- Displaying user or product lists in an admin panel
- Allowing users to filter, sort, and select rows for bulk actions
- Exporting data subsets for reporting or external use
- Showing detailed datasets with interactive actions per row
