# Media Product Card

The Media Product Card is used to quickly present a product with key information and a visual preview, offering users
a clear action and an optional link to learn more.

The Media Product Card is a vertical compact product presentation card designed to provide essential information at a
glance. It displays an image, followed by a title, subtitle, and short description, along with an action button, and
a link.

Use this card when you want to showcase a product. It is suited for product listings, or anywhere users need a quick
overview with a clear path to learn more or take action.
