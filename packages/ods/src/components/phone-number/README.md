# How-to generate flag spritesheet

For now, we use https://github.com/flarik/css-sprite-flags.

You will need to add those that are missing in the repo css:

```scss
'bq': -264px -240px,
'gf': -192px -120px,
'gp': -192px -120px,
'io': -240px -120px,
'pm': -192px -120px,
're': -192px -120px,
'sj': -288px -240px,
'sx': -264px -240px,
'wf': -0px -360px,
'ws': -24px -360px,
'ye': -48px -360px,
'yt': -72px -360px,
'za': -96px -360px,
'zm': -120px -360px,
'zw': -144px -360px,
```

here the workflow used to generate such files.
