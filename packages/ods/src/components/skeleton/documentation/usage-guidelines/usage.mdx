## Usage

### Default

<Canvas sourceState="none">
  <ods-skeleton></ods-skeleton>
</Canvas>

```html
<ods-skeleton></ods-skeleton>
```

### Custom CSS

You can add your own style on the text element using the part `skeleton`.

```html
<ods-skeleton class="my-skeleton">
</ods-skeleton>
<style>
  .my-skeleton::part(skeleton) {
    height: 2em;
    width: 10em;
  }
</style>
```
