## Attributes changes

`color` <img src="https://img.shields.io/badge/new-008000" />

New attribute.

Provides a list of colors using the currently applied theme.

`contrasted` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

You can use the new `TODO` color to obtain the same rendering.

`inline` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

Spinner is now `inline-block` by default.
You can add some style directly on the component if you want to override that.

`mode` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

There was only was mode value, it is now the default rendering.

`size` <img src="https://img.shields.io/badge/updated-00FFFF" />

Has been updated. TODO (see if it's actually the case or do we have the same as before, in that case remove this whole part)

Provides a list of premade spinner sizer

## Migration examples

Contrasted spinner:
```html
<ods-spinner inline contrasted size="sm"></ods-spinner>
<!-- is now -->
<ods-spinner color="TODO" size="sm"></ods-spinner>
```
