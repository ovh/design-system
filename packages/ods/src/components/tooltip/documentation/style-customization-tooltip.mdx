You can update the tooltip content by directly updating the slot element.

Example:
```html
<ods-tooltip position="top-start"
             trigger-id="top-start"
             with-arrow>
  <p class="custom-tooltip">
    Top-start tooltip
  </p>
</ods-tooltip>
<style>
  .custom-tooltip {
    font-style: italic;
  }
</style>
```
