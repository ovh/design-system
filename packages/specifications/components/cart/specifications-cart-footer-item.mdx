## Description
The cart footer item displays a title, a subhead and a price equivalent to a subtotal. Extra information is optional.

<Description>{Specs}</Description>

## Contents

| Name      | Description                           |
|-----------|---------------------------------------|
| `title`   | the item name (displayed as accented) |
| `subhead` | the item context or reference         |
| `price`   | the item option price excluded VAT    |
| `extra`   | extra information like "HT / month"   |

## Tests

**Extra zone**
- should displays the passed content in the extra zone

**Title**
- should display the name of the item

**Subhead**
- should display the specified subhead

**Price**
- should display the subtotal price of the added item
