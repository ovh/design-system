## ODS Starters

You built some application starters ready to use with the most used framework.
These samples are made in order to get started more efficiently.
They integrate `ODS` components and show how to use them.

Classic `React` application
### [@ovhcloud/ods-starter-react-vite](?path=/story/code-starters-react-react-vite--page)
`React` application with `Vite` compiler
### [@ovhcloud/ods-starter-vue-vite](?path=/story/code-starters-vue-vue-vite--page)
`VueJs` application with `Vite` compiler
