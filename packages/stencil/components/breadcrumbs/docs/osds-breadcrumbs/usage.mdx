## Usage

### Sizes
```html
<ods-breadcrumbs size='s'>Small</ods-breadcrumbs>
<ods-breadcrumbs>Default</ods-breadcrumbs>
```

### Variants
```html
<ods-breadcrumbs variant='ghost'>Ghost</ods-breadcrumbs>
<ods-breadcrumbs variant='primary'>Primary</ods-breadcrumbs>
<ods-breadcrumbs variant='secondary'>Secondary</ods-breadcrumbs>
```

### Block
```html
<ods-breadcrumbs block size='s'>Block small</ods-breadcrumbs>
<ods-breadcrumbs block>Block</ods-breadcrumbs>
```

### Square
```html
<ods-breadcrumbs square size='s'>Square small</ods-breadcrumbs>
<ods-breadcrumbs square>Square</ods-breadcrumbs>
```

### Disabled
```html
<ods-breadcrumbs size='s' disabled>Small</ods-breadcrumbs>
<ods-breadcrumbs disabled>Default</ods-breadcrumbs>
<ods-breadcrumbs variant='ghost' disabled>Ghost</ods-breadcrumbs>
<ods-breadcrumbs variant='primary' disabled>Primary</ods-breadcrumbs>
<ods-breadcrumbs variant='secondary' disabled>Secondary</ods-breadcrumbs>
<ods-breadcrumbs block size='s' disabled>Block small</ods-breadcrumbs>
<ods-breadcrumbs block disabled>Block</ods-breadcrumbs>
<ods-breadcrumbs square size='s' disabled>Square small</ods-breadcrumbs>
<ods-breadcrumbs square disabled>Square</ods-breadcrumbs>
```
