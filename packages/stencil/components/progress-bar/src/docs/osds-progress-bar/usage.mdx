## Usage

### Default
<osds-progress-bar value="50"></osds-progress-bar>

```html
<osds-progress-bar value="50"></osds-progress-bar>
```

### With slots
<osds-progress-bar value="50">
  <span slot="before">0%</span>
  <span slot="after">100%</span>
  <span slot="under">Downloading...</span>
</osds-progress-bar>

```html
<osds-progress-bar value="50">
  <span slot="before">0%</span>
  <span slot="after">100%</span>

</osds-progress-bar>
```

### With max value
<osds-progress-bar value="50" max="60"></osds-progress-bar>

```html
<osds-progress-bar value="50" max="60"></osds-progress-bar>
```
