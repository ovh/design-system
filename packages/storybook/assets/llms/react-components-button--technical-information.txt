## Button

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| This component extends all the native [button attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#attributes) . |
| color | `BUTTON_COLOR` | - | `BUTTON_COLOR.primary` | The color preset to use. |
| loading | `boolean` | - | `false` | Whether the component is in loading state, disabling it. |
| size | `BUTTON_SIZE` | - | `BUTTON_SIZE.md` | The size preset to use. |
| variant | `BUTTON_VARIANT` | - | `BUTTON_VARIANT.default` | The variant preset to use. |
