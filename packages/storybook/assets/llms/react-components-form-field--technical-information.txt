## Examples

---

### Default

```jsx
<FormField>
  <Textarea name="textarea" />
</FormField>
```

### Field label

Description:

```jsx
<FormField>
  <FormFieldLabel>
    Description:  </FormFieldLabel>
  <Textarea name="textarea" />
</FormField>
```

### Field sub label

Description- mandatory

```jsx
<FormField>
  <FormFieldLabel>
    Description    <FormFieldLabelSubLabel>
      - mandatory    </FormFieldLabelSubLabel>
  </FormFieldLabel>
  <Textarea name="textarea" />
</FormField>
```

### Helper message

Helper text

```jsx
<FormField>
  <Textarea name="textarea" />
  <FormFieldHelper>
    <Text preset="caption">
      Helper text    </Text>
  </FormFieldHelper>
</FormField>
```

### Error message

Error message

```jsx
<FormField invalid>
  <Textarea name="textarea" />
  <FormFieldError>
    Error message  </FormFieldError>
</FormField>
```
