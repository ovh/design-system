## Examples

---

### Default

```jsx
<Medium
  alt="NASA Computer room"
  src="https://upload.wikimedia.org/wikipedia/commons/b/b9/NASAComputerRoom7090.NARA.jpg"
/>
```

```jsx
/>
```

### Width

```jsx
<Medium

/>
```
