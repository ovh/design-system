## Meter

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| high | `number` | - | `undefined` | The lower numeric bound of the high end of the measured range. |
| low | `number` | - | `undefined` | The upper numeric bound of the low end of the measured range. |
| max | `number` | - | `100` | The upper numeric bound of the measured range. |
| min | `number` | - | `0` | The lower numeric bound of the measured range. |
| optimum | `number` | - | `undefined` | The optimal numeric value. Combined with low and high, it will changes the coloring behaviour. |
| value | `number` | - | `0` | The current value of the meter |
