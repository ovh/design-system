## Spinner

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| This component extends all the native [span attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/span#attributes) . |
| color | `SPINNER_COLOR` | - | `SPINNER_COLOR.primary` | The color preset to use. |
| size | `SPINNER_SIZE` | - | `SPINNER_SIZE.md` | The size preset to use. |

## Enums

-   lg =`"lg"`
-   md =`"md"`
