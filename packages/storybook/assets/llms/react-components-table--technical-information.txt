## Table

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| This component extends all the native [table attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/table#attributes) . |
| size | `TABLE_SIZE` | - | `TABLE_SIZE.md` | The size preset to use. |
| variant | `TABLE_VARIANT` | - | `TABLE_VARIANT.default` | The variant preset to use. |

## Enums

---

### TABLE_SIZE

-   lg =`"lg"`
-   md =`"md"`
-   sm =`"sm"`

### TABLE_VARIANT

-   default =`"default"`
-   striped =`"striped"`
