## Tag

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| This component extends all the native [button attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#attributes) . |
| color | `TAG_COLOR` | - | `TAG_COLOR.information` | The color preset to use. |
| size | `TAG_SIZE` | - | `TAG_SIZE.md` | The size preset to use. |

### Default

```jsx
<Tag>
  My tag
</Tag>
```

### Color

```jsx
<>
  <Tag color="critical">
    Critical  </Tag>
  <Tag color="information">
    Information  </Tag>
  <Tag color="neutral">
    Neutral  </Tag>
  <Tag color="primary">
    Primary  </Tag>
  <Tag color="success">
    Success  </Tag>
  <Tag color="warning">
    Warning  </Tag>
</>
```

### Disabled

```jsx
<Tag disabled>
  My tag
</Tag>
```

### Size

```jsx
<>
  <Tag size="md">
    MD tag  </Tag>
  <Tag size="lg">
    LG tag  </Tag>
</>
```
