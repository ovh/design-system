## Text

---

| Property | Type | Required | Default value | Description |
| --- | --- | --- | --- | --- |
| This component extends all the native [text elements attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/label#attributes) . |
| as | `string` | - | `undefined` | Pass a HTML tag you may want to use instead of the preset one. Useful for example to apply heading style to a non-heading element. |
| disabled | `boolean` | - | `undefined` | Whether the text is displayed in a disabled context. |
| preset | `TEXT_PRESET` | - | `TEXT_PRESET.paragraph` | The text preset to use. |

### Caption preset in <table>

My table title
| Person | Age |
| --- | --- |
| Chris | 22 |

```jsx
<table
  style={{
    border: '2px solid rgb(140 140 140)',
    borderCollapse: 'collapse'
  }}
>
  <caption
    style={{
      captionSide: 'bottom'
    }}
  >
    <Text preset="caption">
      My table title    </Text>
  </caption>
  <thead>
    <tr>
      <th scope="col">
        Person      </th>
      <th scope="col">
        Age      </th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">
        Chris      </th>
      <td>
        22      </td>
    </tr>
  </tbody>
</table>
```

### Preset style on a different HTML element

I am a <span> using the heading5 preset style.

```jsx
<Text
  as="span"
  preset="heading-5"
>
  {`I am a <span> using the heading5 preset style.`}
</Text>
```
