## basic

Here is the easier way to use the overlay system in a vanilla way.
The main principle is to declare an HTML element like a div and setting
it as the `anchor` element of the surface.
To do that, you need to have access to the trigger html element and
the declared `surface` element with all the content.

Don't hesitate to use the `RTL` feature. The overlay will adapt to this HTML instruction:

```html
<div id="basic-container-1" dir="rtl">
  <div id="basic-trigger-1" onclick="basic1Toggle()">My trigger</div>
  <ocdk-surface id="basic-surface-1">My surface</ocdk-surface>
</div>
```
