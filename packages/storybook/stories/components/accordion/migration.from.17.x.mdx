# Accordion - migrate from v17 to v18
----

## Attributes changes

`color` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`contrasted` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`disabled ` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`opened ` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isOpen` attribute to obtain the same behavior.

`size ` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

## Migration examples

Without Color:
```html
<osds-accordion color='information'></osds-accordion>

<!-- is now -->

<ods-accordion></ods-accordion>
```

Without Contrasted:
```html
<osds-accordion contrasted></osds-accordion>

<!-- is now -->

<ods-accordion></ods-accordion>
```

Disabled:
```html
<osds-accordion disabled></osds-accordion>

<!-- is now -->

<ods-accordion is-disabled></ods-accordion>
```

Opened:
```html
<osds-accordion opened></osds-accordion>

<!-- is now -->

<ods-accordion is-open></ods-accordion>
```

Without Size:
```html
<osds-accordion size='md'></osds-accordion>

<!-- is now -->

<ods-accordion></ods-accordion>
```
