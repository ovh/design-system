## Usage

A **Card** can be used to:

* display a wide variety of content such as a [**Text**](?path=/docs/ods-components-text--documentation), a [**Link**](?path=/docs/ods-components-link--documentation) or other elements like a [**Button**](?path=/docs/ods-components-button--documentation)
* emphasize important content
* visually group related content
* guide the user toward actions
* provide the option to navigate to more detailed content

Several elements can be used in a **Card**:

* A title
* A description
* A [**Button**](?path=/docs/ods-components-button--documentation)
* A [**Medium**](?path=/docs/ods-components-medium--documentation)
* Extra information

All these elements are not mandatory in every **Card**.

## Placement

**Cards** have an automatic width, but they can widen to match the container.

**Cards** that are strongly related can be grouped in a layout group. This group can flow horizontally left to right and/or vertically top to bottom.
