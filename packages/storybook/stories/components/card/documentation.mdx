<Heading label="Usage" level={ 2 } />

A **Card** can be used to:

* display a wide variety of content such as a <StorybookLink kind="ODS Components/Text" label="Text" story="Documentation" />, a <StorybookLink kind="ODS Components/Link" label="Link" story="Documentation" /> or other elements like a <StorybookLink kind="ODS Components/Button" label="Button" story="Documentation" />
* emphasize important content
* visually group related content
* guide the user toward actions
* provide the option to navigate to more detailed content

Several elements can be used in a **Card**:

* A title
* A description
* A <StorybookLink kind="ODS Components/Button" label="Button" story="Documentation" />
* A <StorybookLink kind="ODS Components/Medium" label="Medium" story="Documentation" />
* Extra information

All these elements are not mandatory in every **Card**.

<Heading label="Dos & Don'ts" level={ 3 } />

<BestPractices

  ]}
/>

<Heading label="Placement" level={ 2 } />

**Cards** have an automatic width, but they can widen to match the container.

**Cards** that are strongly related can be grouped in a layout group. This group can flow horizontally left to right and/or vertically top to bottom.
