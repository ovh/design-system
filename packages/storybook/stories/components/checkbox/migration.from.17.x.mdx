# Checkbox - migrate from v17 to v18
----

## Usages changes

The previous components `checkbox` and `checkbox-button` have now been included in `checkbox`.

There is no longer a `checkbox` component to make clickable Element.

## Attributes changes

`checked` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isChecked` attribute to obtain the same behavior.

`checking` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`color` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`disabled` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`hasFocus` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

You can use the event `odsFocus` and `odsBlur` for have the same behavior.

`indeterminate` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isIndeterminate` attribute to obtain the same behavior.

`interactive` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`label` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Checkbox can now use HTML Label. Like in the example.

`updating` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

We don't think this kind of attribute was necessary anymore.

This design feature has been removed in order to match the new styling of the component.

We don't think this kind of callback was necessary anymore.

`save` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

We don't think this kind of callback was necessary anymore.

## Migration examples

Checked checkbox:
```html
<osds-checkbox checked>
  <osds-checkbox-button></osds-checkbox-button>
</osds-checkbox>

<!-- is now -->

<ods-checkbox is-checked></ods-checkbox>
```

Disabled checkbox:
```html
<osds-checkbox disabled>
  <osds-checkbox-button></osds-checkbox-button>
</osds-checkbox>

<!-- is now -->

<ods-checkbox is-disabled></ods-checkbox>
```

Indeterminate checkbox:
```html
<osds-checkbox>
  <osds-checkbox-button indeterminate></osds-checkbox-button>
</osds-checkbox>

<!-- is now -->

<ods-checkbox is-indeterminate></ods-checkbox>
```

Label checkbox:
```html
<osds-checkbox>
  <osds-checkbox-button>
     <span slot="end">
      Checkbox label
    </span>
  </osds-checkbox-button>
</osds-checkbox>

<!-- is now -->

<ods-checkbox input-id="migration-label"></ods-checkbox>
<label for="migration-label">Checkbox label</label>
```
