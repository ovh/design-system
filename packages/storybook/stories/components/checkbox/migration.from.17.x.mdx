<Heading label="Checkbox - migrate from v17 to v18" level={ 1 } />

<Heading label="Usage changes" level={ 2 } />

The previous components `checkbox` and `checkbox-button` have now been included in `checkbox`.

There is no longer a `checkbox` component to make clickable Element.

<Heading label="Attributes changes" level={ 2 } />

`checked` <Badge color="information" size="sm">Updated</Badge>

Has been updated.

You can use the new `isChecked` attribute to obtain the same behavior.

`checking` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`color` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`disabled` <Badge color="information" size="sm">Updated</Badge>

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`hasFocus` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

You can use the event `odsFocus` and `odsBlur` for have the same behavior.

`indeterminate` <Badge color="information" size="sm">Updated</Badge>

Has been updated.

You can use the new `isIndeterminate` attribute to obtain the same behavior.

`interactive` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`label` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Checkbox can now use HTML Label. Like in the example.

`updating` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

We don't think this kind of attribute was necessary anymore.

`size` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`afterSave` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

We don't think this kind of callback was necessary anymore.

`beforeSave` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

We don't think this kind of callback was necessary anymore.

`save` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

We don't think this kind of callback was necessary anymore.
