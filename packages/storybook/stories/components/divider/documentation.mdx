<Heading label="Usage" level={ 2 } />

**Divider** is used to separate content on a page.

It brings clarity to a layout by dividing content in proximity.

<Heading label="Dos & Don'ts" level={ 3 } />

<BestPractices
  donts={[
    '- Use a Divider with a line separator above a header / a section title',
    '- Use Divider sparingly on a page',
    '- Overuse Divider on a page, so it won\'t create unnecessary noise',
  ]}
  dos={[
    '- Use Divider to separate two elements',
    '- Use a Divider as a substitute for styled semantic sectioning HTML (for example borders)',
    '- A Divider with a line separator might be used below a header / a section title',
  ]}
/>

<Heading label="Placement" level={ 2 } />

The **Divider** has to be used to create a separation between two elements and to separate the contents of a page.

The **Divider** can also be used inside components to create separation between a set of sections.
