# Divider - migrate from v17 to v18
----
## Usage changes

The divider is a visual separator and shouldn't be used as a vertical separating measure.

## Attributes changes

`color`  <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

The 2 possible values are now `light` and `dark`.

`contrasted` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Color attribute can be use with `dark` for contrasted content.

`separator ` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Divider is always shown by default.

`size ` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

`spacing ` <ods-badge size="sm" color="success" label="New"></ods-badge>

It allows to control the height in pixel of the divider.

## Migration examples

Color:
```html
<osds-divider color="primary"></osds-divider>

<!-- is now -->

<ods-divider class="custom"></ods-divider>

<style>
  .custom {
    background-color: var(--ods-color-primary-100);
  }
</style>
```

Contrasted:
```html
<osds-divider contrasted></osds-divider>

<!-- is now -->

<ods-divider color="dark"></ods-divider>
```

Separator:
```html
<osds-divider separator="false"></osds-divider>

<!-- is now -->

<ods-divider class="custom"></ods-divider>

<style>
  .custom {
    background-color: transparent;
  }
</style>
```

Size:
```html
<osds-divider size="1"></osds-divider>

<!-- is now -->

<ods-divider spacing="2"></ods-divider>
```
