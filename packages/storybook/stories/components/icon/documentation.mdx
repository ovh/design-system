## Usage

**Icons** enhance the usability and bring clarity to users by reducing the cognitive load.

**Icons** can be used in standalone [**Button**](?path=/docs/ods-components-actions-button--documentation) and can be usually associated with some sort of action.

**Icons** can be used as navigational elements as well.

### Dos & Don'ts

<Markdown>
  { `
| ✅ Do                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| - Give an equivalent text as aria-name to the Icon                                                                                                                             |
| - Assure a good contrast between the background and Icon colors                                                                                                                |
| - Center-align when **Icons** are next to [**Text**](?path=/docs/ods-components-content-tag--documentation)                                                                    |
| - When **Icons** are next to [**Text**](?path=/docs/ods-components-content-tag--documentation), use the same color for pairing them                                            |
` }
</Markdown>
<Markdown>
  { `
| ❌ Don't                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Use for decorative purpose                                                                                                                                                     |
| - Use the same **Icon** for different purposes to avoid user's confusion                                                                                                         |
| - Use similar shades of the same color for background and **Icon** colors                                                                                                        |
| - Baseline-align **Icons** when they are next to [**Text**](?path=/docs/ods-components-content-tag--documentation)                                                               |
| - When **Icons** are next to [**Text**](?path=/docs/ods-components-content-tag--documentation), don't use different colors for both of them                                      |
` }
</Markdown>

## Placement

**Icons** draw attention to important information or action, so it has to be thoughtfully inserted in a page.

It also can be used in other components where it is adapted for a specific purpose.

## Behavior

**Icons** can stand alone or be included in a [**Button**](?path=/docs/ods-components-actions-button--documentation) or other components depending on the usage.
