<Heading label="Attributes changes" level={ 2 } />

`ariaName` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Use native <ExternalLink href="https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-label">aria-label</ExternalLink> instead.

`color` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Icons are now font, they will by default have the same color as the current text.

You can override this using a CSS class on the component directly.

`contrasted` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Icons are now font, they will by default have the same color as the current text.

`hoverable` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Interactive icon will be handled specifically by components that implements those interactions.

`size` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Icons are now font, they will by default have the same size as the current text.

You can override this using a CSS class on the component directly.

> **Warning:**
> Icons have been resized to fit entirely their bounding box.
> Check your current icons and update their size accordingly.
