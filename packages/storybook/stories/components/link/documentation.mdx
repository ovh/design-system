2. [**Icon**](?path=/docs/ods-components-icon--documentation) (optional): Indicates the destination or type of content being.

## Usage

**Links** are used as navigational elements.

They are also used when linking to another document. In that case, **Link** description should contain type and size of document.

### When to use a **Link** or a [**Button**](?path=/docs/ods-components-button--documentation):

They have different purposes so, make sure you are accurately using them.

[**Button**](?path=/docs/ods-components-button--documentation) is used for actions while **Link** is used for navigation.

### Dos & Don'ts

<Markdown className="documentation-do-table">
  { `
| ✅ Do                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| - Use a **Link** to redirect user to a new page, section or website.                                                                                                           |
| - Make a **Link** description self-explanatory to describe where user will be send after following this **Link**                                                               |
| - Write short key words as **Link** description                                                                                                                                |
| - Ensure users are aware of a **Link** containing a document by providing its type and size                                                                                    |
` }
</Markdown>

<Markdown className="documentation-dont-table">
  { `
| ❌ Don't                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Use a **Link** as a [**Button**](?path=/docs/ods-components-button--documentation) for actions such as 'save', 'cancel' or 'submit'                                    |
| - Use "Click Here" or "Here" as description for a **Link**                                                                                                                       |
| - Write a full long sentence as **Link** description                                                                                                                             |
| - Use different color for **Link** and the optional [**Icon**](?path=/docs/ods-components-icon--documentation) when you add one                                          |
` }
</Markdown>

## Placement

A **Link** can stand alone on a page, or it can be placed within a sentence or a paragraph.

An optional [**Icon**](?path=/docs/ods-components-icon--documentation) can be placed next to the **Link**.

## Behavior

**Link** component can be hovered, focused, clicked and disabled.

When disabled, users cannot focus nor click on the **Link**.

On hover, **Link** is underlined and its color changes.

When users clicks anywhere on the **Link**, even its optional [**Icon**](?path=/docs/ods-components-icon--documentation), they are redirected to the expected page or section.
