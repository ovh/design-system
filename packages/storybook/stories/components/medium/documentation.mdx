## Usage

**Medium** is used to display assets with an alternative [**Text**](?path=/docs/ods-components-text--documentation) description.

When the asset is an image, it should be mostly presentation to draw user's attention visually.

### Dos & Don'ts

<Markdown className="documentation-do-table">
  { `
| ✅ Do                                                                                                                                                                           |

<Markdown className="documentation-dont-table">

| ❌ Don't                                                                                                                                                                          |
