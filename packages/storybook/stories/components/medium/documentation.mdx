**Medium** only has a single main content.

## Usage

**Medium** is used to display assets with an alternative [**Text**](?path=/docs/ods-components-text--documentation) description.

When the asset is an image, it should be mostly presentation to draw user's attention visually.

### Dos & Don'ts

<Markdown>
  { `
| ✅ Do                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Provide captions or descriptions for media content where applicable.                                                                                                         |
` }
</Markdown>
<Markdown>
  { `
| ❌ Don't                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Avoid assets with text in them since screen readers won't be able to read the words.                                                                                           |
| - Avoid unnecessary decoration                                                                                                                                                   |
` }
</Markdown>
