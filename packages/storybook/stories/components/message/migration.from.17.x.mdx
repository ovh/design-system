# Message - migrate from v17 to v18
----

## Attributes changes

`contrasted` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

You can use the style customization to achieve the same result, until a new color get validated by the design team.

`inline` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Message is inline by default. If you need to change this, you can use the style customization.

`removable` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Message is always removable.

`type` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Color attribute will condition which icon is displayed.

`variant` <ods-badge size="sm" color="success" label="New"></ods-badge>

New attribute.

Allow you to choose between default or light variant, a white version for lighter notification but different from contrasted color.

## Migration examples

Basic tag:
```html
<osds-message>
  My message
</osds-message>

<!-- is now -->

<ods-message>
  My message
</ods-message>
```
