# Message - migrate from v17 to v18
----

## Attributes changes

`contrasted ` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

You can use the style customization to achieve the same result, until a new color get validated by the design team.

`inline` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

Message is inline by default. If you need to change this, you can use the style customization.

`removable` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

Message is always removable.

`type` <img src="https://img.shields.io/badge/removed-FF0000" />

Has been removed.

Color attribute will condition which icon is displayed.

`variant` <img src="https://img.shields.io/badge/new-008000" />

New attribute.

Allow you to choose between default or light variant, a white version for lighter notification but different from contrasted color.

## Migration examples

Basic tag:
```html
<osds-message>
  My message
</osds-message>

<!-- is now -->

<ods-message>
  My message
</ods-message>
```
