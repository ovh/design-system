# Modal - migrate from v17 to v18
----

## Attributes changes

`color` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

Modal's color options have been restricted to the options in `ODS_MODAL_COLORS`.

`dismissible` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isDismissible` attribute to obtain the same behavior.

`headline` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Modal does not provide a headline attribute anymore.
You can use the content slot to recreate the desired headline.

`masked` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isOpen` attribute to obtain the same behavior.

## Migration examples

Color modal:
```html
<osds-modal color={ODS_THEME_COLOR_INTENT.information}></osds-modal>

<!-- is now -->

<ods-modal color={ODS_MODAL_COLOR.information}></ods-modal>
```

Dismissible modal:
```html
<osds-modal dismissible="true"></osds-modal>

<!-- is now -->

<ods-modal is-dismissible="true"></ods-modal>
```

Headline modal:
```html
<osds-modal headline="Hello, world!"></osds-modal>

<!-- is now -->

<ods-modal>
  <ods-text preset="heading-3">Hello, world!</ods-text>
</ods-modal>
```

Masked modal:
```html
<osds-modal masked="true"></osds-modal>

<!-- is now -->

<ods-modal is-open="false"></ods-modal>
```
