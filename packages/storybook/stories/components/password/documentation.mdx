## Anatomy

![Component anatomy](components/password/anatomy.png "Component anatomy")

1. **Placeholder**: it provides hints or example of what user should type
2. **Container**: it is a container where the user can type in
3. **Clear action**: a `close` icon [**Button**](?path=/docs/ods-components-actions-button--documentation) so that user can clear the [**Input**](?path=/docs/ods-components-form-input--documentation) content (optional)
4. **Show/hide action**: an `eye` icon [**Button**](?path=/docs/ods-components-actions-button--documentation) to toggle between masked or visible [**Input**](?path=/docs/ods-components-form-input--documentation) content

## Usage

A **Password** is mainly used to let users enter a password or secret information.

Entered password can be masked or visible as plain text when users need to check what they have entered.

It can also be used for displaying tokens.

### Dos & Don'ts

<Markdown>
  { `
| ✅ Do                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| - Use a **Password** when user has to enter sensitive data in the field                                                                                                        |
| - Use read-only **Password** for tokens with copyable action enabled                                                                                                           |
` }
</Markdown>
<Markdown>
  { `
| ❌ Don't                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Use a **Password** when you need the text users enter to be only visible. You should use an [**Input**](?path=/docs/ods-components-form-input--documentation) instead          |
` }
</Markdown>

## Placement

By default, the **Password** content is left-aligned in its container.

## Behavior

By default, the **Password** content is masked.

**Password** can be used as read-only (especially for tokens).

The password masking toggled using show/hide action is permanent. Users have to click again to show/hide the [**Input**](?path=/docs/ods-components-form-input--documentation) content.

It inherits events from [**Input**](?path=/docs/ods-components-form-input--documentation) component.
