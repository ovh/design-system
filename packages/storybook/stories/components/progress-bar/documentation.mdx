<IdentityCard aliases={ ['Progress', 'Progress Loader', 'Preloader', 'Loading Bar', 'Progress Indicator'] }

  The **Progress Bar** component is used to indicate the progress of a task or process to users. It visually
  represents the completion percentage, providing users with feedback on the status of ongoing operations, such as
  <StorybookLink kind="ODS Components/Form elements/File Upload" label="File upload" story="Documentation" />, downloads, or form submissions.
</IdentityCard>

<Heading label="Anatomy" level={ 2 } />

![Component anatomy](components/progress-bar/anatomy.png "Component anatomy")

1. **Track**: displays the visual range of progression, from a starting to an ending position
2. **Filled track**: represents the visual progression value within the **Progress Bar**.

<Heading label="Usage" level={ 2 } />

**Progress Bar** component is only used in process progression, such as submitting, uploading or saving items.

<Heading label="Dos & Don'ts" level={ 3 } />

<BestPractices
  donts={[
    '- Use Progress Bar for data visualization',
    '- Use all three starting value, ending value and progress value labels at the same time',
  ]}
  dos={[
    <span>- Use Progress Bar for quantifiable loading; if not, please use <StorybookLink kind="ODS Components/Spinner" label="Spinner" story="Documentation" /> for complex content or <StorybookLink kind="ODS Components/Skeleton" label="Skeleton" story="Documentation" /> for <StorybookLink kind="ODS Components/Text" label="Text" story="Documentation" />-only content</span>,
  ]}
/>

<Heading label="Placement" level={ 2 } />

**Progress Bar** is usually centered in its container, and can be stretched to match the container width if necessary.

<Heading label="Behavior" level={ 2 } />

**Progress Bar** is filled from the minimum to the maximum value, depending on the progress described.
