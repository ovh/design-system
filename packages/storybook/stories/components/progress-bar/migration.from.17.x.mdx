# ProgressBar - migrate from v17 to v18
----

## Slot changes

`end` <img src="https://img.shields.io/badge/remove-FF0000" />

Has been removed.

You can now add any text directly after the component instead of using the slot.

## Migration examples

Slot end progress bar:
```html
<osds-progress-bar value="50">
  <span slot="end">50%</span>
</osds-progress-bar>

<!-- is now -->

<ods-progress-bar value="50"></ods-progress-bar><span>50%</span>
```
