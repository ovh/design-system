# Radio - migrate from v17 to v18
----

## Attributes changes Radio

`checked` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isChecked` attribute to obtain the same behavior.

`checking` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`disabled` <ods-badge size="sm" label="Updated"></ods-badge>

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`label` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

Radio can now use HTML Label. Like in the example.

`afterSave` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

We don't think this kind of callback was necessary anymore.

`beforeSave` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

We don't think this kind of callback was necessary anymore.

`save` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

We don't think this kind of callback was necessary anymore.

## Attributes changes Radio group

`required` <ods-badge size="sm" label="Updated"></ods-badge>

`color` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

You can use the CSS to obtain the same rendering.

`hasFocus` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

You have now an event to know when the component is focused.

`size` <ods-badge size="sm" color="critical" label="Removed"></ods-badge>

Has been removed.

You can use the CSS to obtain the same rendering.

## Migration examples

Disabled radio:
```html
<osds-radio disabled>
  <osds-radio-button color='primary' size='sm'></osds-radio-button>
</osds-radio>

<!-- is now -->

<ods-radio is-disabled></ods-radio>
```

Label radio:
```html
<osds-radio-button>
  <span slot='end'>Radio label</span>
</osds-radio-button>

<!-- is now -->

<ods-radio input-id="migration-label" name="name" value="migration"></ods-radio>
<label for="migration-label">Radio label</label>
```

Group radio:
```html
<osds-radio-group>
    <osds-radio value="A">
        <osds-radio-button color='primary' size='sm'></osds-radio-button>
    </osds-radio>
    <osds-radio value="B">
        <osds-radio-button color='primary' size='sm'></osds-radio-button>
    </osds-radio>
    <osds-radio value="C">
        <osds-radio-button color='primary' size='sm'></osds-radio-button>
    </osds-radio>
</osds-radio-group>

<!-- is now -->

<ods-radio value="A" name="group-radio"></ods-radio>
<ods-radio value="B" name="group-radio"></ods-radio>
<ods-radio value="C" name="group-radio"></ods-radio>

```
