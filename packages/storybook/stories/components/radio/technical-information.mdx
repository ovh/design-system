## Overview

**Radio** are used when, facing a list of two or more options, the user must select only one.

The previous components `radio`, `radio-button` and `radio-group` have now been included in `radio`

The behavior of the old `radio-group` is now automatic, giving the radio the same name as the native implementation.

There is no longer a `radio` component to make clickable Element.
