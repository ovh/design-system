## Usage

A **Skeleton** can be used in a variety of contexts such as in [**Card**](?path=/docs/ods-components-card--documentation), lists, and [**Table**](?path=/docs/ods-components-table--documentation) content.

### Dos & Don'ts

<Markdown className="documentation-do-table">
  { `
| ✅ Do                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| - Prefer using a **Skeleton** for dynamic content except it is necessary for your use case                                                                                     |
| - Use a **Skeleton** if content loading is going to take a few seconds                                                                                                         |
| - Use when the page or a part of the page is loading or updating                                                                                                               |
` }
</Markdown>

<Markdown className="documentation-dont-table">
  { `
| ❌ Don't                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| - Avoid using a **Skeleton** for static content                                                                                                                                  |
| - Use for interactive elements                                                                                                                                                   |
` }
</Markdown>

## Placement

A **Skeleton** can be placed whenever needed to indicate a content is loading.

## Behavior

The **Skeleton** component has a linear animation to show to indicates a content is loading.
