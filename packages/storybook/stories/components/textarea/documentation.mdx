<Heading label="Usage" level={ 2 } />

Use the **Textarea** component to allow users to enter multi-line text in forms, such as comments, messages, descriptions, or other detailed information.

Ensure that the **Textarea** is appropriately sized for the expected input and provides clear guidance on what information is needed.

<Heading label="Placement" level={ 2 } />

**Textarea** is used in forms where the expected user input text would be long.

<Heading label="Behavior" level={ 2 } />

**Textarea** can be hovered, focused, disabled or read only.

**When disabled**, the component cannot be focused nor clicked.

When the user types an amount of **text longer than Textarea** container itself, a scrollbar is displayed.

**Textarea** is optionally resizable: user can expand its container.

An event is triggered when the **Textarea** value changes, and on user focusing in and out of the component.
