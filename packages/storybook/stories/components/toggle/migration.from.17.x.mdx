<Heading label="Attributes changes" level={ 2 } />

`checked` <Badge color="information" size="sm">Updated</Badge>

Has been updated.

You can use the new `value` attribute to obtain the same behavior.

`checking` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`contrasted` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`color` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`disabled` <Badge color="information" size="sm">Updated</Badge>

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`interactive` <Badge color="critical" size="sm">Removed</Badge>

Has been removed.

Toggle is now always interactive.

`isRequired` <Badge color="success" size="sm">New</Badge>

New attribute (optional).

Defines if the toggle is required in a form.

`name` <Badge color="success" size="sm">New</Badge>

New attribute.

You can use the new 'name' attribute to identify your OdsToggle within a form.

`withLabel` <Badge color="success" size="sm">New</Badge>

New attribute (optional).

Will add the visual labels `ON/OFF`.
