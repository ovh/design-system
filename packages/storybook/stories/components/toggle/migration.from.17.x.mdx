<Heading label="Attributes changes" level={ 2 } />

`checked` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `value` attribute to obtain the same behavior.

`checking` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`contrasted` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`color` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

This design feature has been removed in order to match the new styling of the component.

`disabled` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`interactive` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Toggle is now always interactive.

`isRequired` <OdsBadge color="success" label="New" size="sm" />

New attribute (optional).

Defines if the toggle is required in a form.

`name` <OdsBadge color="success" label="New" size="sm" />

New attribute.

You can use the new 'name' attribute to identify your OdsToggle within a form.

`withLabel` <OdsBadge color="success" label="New" size="sm" />

New attribute (optional).

Will add the visual labels `ON/OFF`.
