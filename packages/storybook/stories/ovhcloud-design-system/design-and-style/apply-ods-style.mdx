<Heading label="Apply ODS Style" level={ 1 } />

After installing ODS (following the <StorybookLink label="Get Started" title={ HOME_TITLE.getStarted } /> documentation),
you'll be able to use every components with the correct style.

For the non-component related style, you need to update your app accordingly.

<Heading label="Use the ODS font" level={ 2 } />

ODS components does not enforce any font by default (minus a few exception like `ods-code`).

To use one of the ODS fonts in your app, add the expected variables to the top of your app:

<Source code={`
html {
  font-family: var(--ods-font-family-default);
}
`}
        dark="true"
        language="css" />

<Heading label="Use the ODS design tokens" level={ 2 } />

All ODS components are using common design tokens, that are accessible through <OdsLink href="https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties" label="CSS variables" target="_blank" />.

If you want to use any of those on your own components, you can refer to the exhaustive list on the <OdsLink href="https://www.figma.com/design/tIKzHa5KvHHyosgIgyBswB/Design-Tokens?m=auto&node-id=0-1&t=s57Qt3pa7WuFWKEh-1" label="Figma dedicated page" target="_blank" />.

<Source code={`
.my-custom-title {
  color: var(--ods-color-text);
  ...
}
`}
        dark="true"
        language="css" />
