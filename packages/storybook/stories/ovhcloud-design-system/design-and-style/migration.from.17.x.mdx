# Style - migrate from v17 to v18
----

## Component tokens
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **component-specific spacing tokens** (ex: **`--ods-size-button-*`**, **`--ods-toggle-*`**) have been removed, their values are now directly applied on components.

## Color tokens
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **primitive color tokens** (ex: **`--ods-color-blue-500`**) have been removed, now colors can only be accessible using alias tokens (see corresponding table below).
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **contrasted primitive color tokens** (ex: **`--ods-color-blue-500`**) have been removed, until new designs and specifications gets validated for having contrasted states on components.
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **`--ods-color-*-050`** and **`--ods-color-*-050-contrasted`** have been removed.
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **`--ods-color-*-list`** tokens have been removed, as no usage for them in integration.

## Size tokens
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **global size tokens** (ex: **`--ods-size-*`**) have been removed (see corresponding table below).
* <img src="https://img.shields.io/badge/removed-FF0000" /> Global sizing ratio (**`--ods-size-ratio`**) which was applied to every spacing concept has been removed due to no use.
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **breakpoints** (**`--ods-breakpoint-*`**) has been removed, their values are now directly applied on components (see corresponding table below).
* <img src="https://img.shields.io/badge/removed-FF0000" /> All **spacing concept tokens** (**`--ods-size-stack-*`**) has been removed due to no use.

* <img src="https://img.shields.io/badge/removed-FF0000" /> All **typography tokens** (ex: **`--ods-typography-*`**) have been removed, their values are now directly applied on components.
* <img src="https://img.shields.io/badge/removed-FF0000" /> Global font sizing ratio (**`--ods-typo-size-ratio`**) which was applied to every spacing concept has been removed due to no use.
