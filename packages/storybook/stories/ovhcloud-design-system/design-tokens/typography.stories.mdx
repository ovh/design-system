  # Typography

  A typography is defined by different levels, like `heading` or `caption`.
  This part of the system manage the CSS properties related to the fonts like `font-size`, `font-family`, `font-weight`.

  Typography levels:

  | level      | description                                                            |
  |------------|------------------------------------------------------------------------|
  | heading    | the different title to use in the application just like `H1`, `H2`,... |
  | subheading | subtitles                                                              |
  | body       | the main typography to be used in all text                             |
  | caption    | typography to be used for the captions                                 |
  | button     | specific typography for the buttons                                    |

  A specific typography level can be set for the button as it has a specific typography.

  ## Font-family
  Source Sans Pro is the main font used in the OVHcloud universe.

  The related fallback fonts are:
  `"Source Sans Pro", "Trebuchet MS", Arial, "Segoe UI", sans-serif`
