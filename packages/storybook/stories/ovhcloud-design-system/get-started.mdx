<Meta title="OVHcloud Design System/Get Started" />

<Heading label="Get Started" level={ 1 } />

OVHcloud Design System is a set of reusable React UI components to create a consistent user experience across OVHcloud pages and products, acting as a single source of truth.

All OVHcloud Design System packages are available on <OdsLink href="https://www.npmjs.com" target="_blank">NPM</OdsLink>.

<Heading label="Installation" level={ 2 } />

Include **ODS** in your projects like any other npm/yarn module:

<Source code="npm i --save-exact @ovhcloud/ods-react @ovhcloud/ods-themes"
        dark="true"
        language="bash" />
or
<Source code="yarn add --exact @ovhcloud/ods-react @ovhcloud/ods-themes"
        dark="true"
        language="bash" />

<Heading label="Usage" level={ 2 } />

<Source code={`
import { OdsButton } from '@ovhcloud/ods-react';

const App = () => {
  return (
    <OdsButton>
      Some button
    </OdsButton>
  );
};
`}
        dark="true"
        language="tsx" />

Components are compatible with React v18.2+.

<Heading label="Import components constants" level={ 2 } />

All enums, interfaces, events are available directly from the `ods-react` lib

<Source code={ `import { ODS_BUTTON_SIZE, type OdsPaginationEvent } from '@ovhcloud/ods-react';` }
        dark="true"
        language="typescript" />

<Heading label="Import theme" level={ 2 } />

You need to import a theme to display all the components correctly.
It defines all the colors, typographies, etc...

via ES import:

<Source code="import '@ovhcloud/ods-themes/default';"
        dark="true"
        language="typescript" />

via Sass import:

<Source code="@import '@ovhcloud/ods-themes/default';"
        dark="true"
        language="css" />
