<Heading label="Get Started" level={ 1 } />

OVHcloud Design System is a set of reusable UI components to create a consistent user experience across OVHcloud pages and products, acting as a single source of truth.

We provide agnostic Web components and their React, Vue wrapped versions.

All OVHcloud Design System packages are available on <OdsLink href="https://www.npmjs.com" label="NPM" target="_blank" />.

<Heading label="Installation" level={ 2 } />

Include **ODS** in your projects like any other npm/yarn module:

<Source code="npm install @ovhcloud/ods-components @ovhcloud/ods-themes"
        dark="true"
        language="bash" />
or
<Source code="yarn add @ovhcloud/ods-components @ovhcloud/ods-themes"
        dark="true"
        language="bash" />

<Heading label="Usage" level={ 2 } />

<Heading label="Web components (JavaScript)" level={ 3 } />

There are two different ways of using ODS web component, using lazy-loading or bundle.

<Heading label="Lazy-loading" level={ 4 } />

To use lazy-loading, just include in your project the loader script, it will take care of loading each component resources on-demand.

<Source code={`
import { defineCustomElements } from '@ovhcloud/ods-components/dist/loader';
defineCustomElements();
`}
        dark="true"
        language="typescript" />

<Heading label="Bundle" level={ 4 } />

If you prefer to use each component directly, you can import them separately.
This is recommended when working on projects bundled (using webpack, rollup, ...) as it is more optimized for tree-shaking.

<Source code={`
import { defineCustomElement as defineButton } from '@ovhcloud/ods-components/dist/components/ods-button';
defineButton();
`}
        dark="true"
        language="typescript" />

<Heading label="React components" level={ 3 } />

To use a React version of an ODS component, you need to import it from the `react` sub directory.

<Source code={`
import { OdsButton } from '@ovhcloud/ods-components/react';

const App = () => {
  return (
    <OdsButton label="Some button">
    </OdsButton>
  );
};
`}
        dark="true"
        language="tsx" />

Components are compatible with React v16.8+.

<Heading label="Vue components" level={ 3 } />

To use a Vue version of an ODS component, you need to import it from the `vue` sub directory.

<Source code={`
<template>
  <OdsButton label="Some Button">
  </OdsButton>
</template>

<script lang="ts">
  import { defineComponent } from 'vue';
  import { OdsButton } from '@ovhcloud/ods-components/vue';

  export default defineComponent({
    name: 'App',
    components: {
      OdsButton,
    },
  });
</script>
`}
        dark="true"
        language="tsx" />

Components are compatible with Vue 3.

<Heading label="Import components constants" level={ 2 } />

All enums, interfaces, events are available directly from the `ods-components` lib

<Source code={`
import type { OdsPaginationEvent } from '@ovhcloud/ods-components'
import { ODS_BUTTON_SIZE, ODS_BUTTON_SIZES } from '@ovhcloud/ods-components';
`}
        dark="true"
        language="typescript" />

<Heading label="Import theme" level={ 2 } />

You need to import a theme to display all the components correctly.
It defines all the colors, typographies, etc...

via ES import:

<Source code="import '@ovhcloud/ods-themes/default';"
        dark="true"
        language="typescript" />

via Sass import:

<Source code="@import '@ovhcloud/ods-themes/default';"
        dark="true"
        language="css" />
