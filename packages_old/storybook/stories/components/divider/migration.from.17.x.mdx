<Heading label="Divider - migrate from v17 to v18" level={ 1 } />

<Heading label="Usage changes" level={ 2 } />

The divider is a visual separator and shouldn't be used as a vertical separating measure.

<Heading label="Attributes changes" level={ 2 } />

`color`  <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

The 2 possible values are now `light` and `dark`.

`contrasted` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Color attribute can be use with `dark` for contrasted content.

`separator ` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Divider is always shown by default.

`size ` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

`spacing ` <OdsBadge color="success" label="New" size="sm" />

It allows to control the height in pixel of the divider.

<Heading label="Migration examples" level={ 2 } />

<Source code={`

  }

        dark="true"
