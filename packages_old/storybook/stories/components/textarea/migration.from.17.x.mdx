<Heading label="Attributes changes" level={ 2 } />

`color ` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Form components does not provide color variant anymore.
You can use custom CSS if you need to customize the component.

`contrasted` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Form components does not provide color variant anymore.
You can use custom CSS if you need to customize the component.

`disabled` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `isDisabled` attribute to obtain the same behavior.

`error` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `hasError` attribute to obtain the same behavior.

`errorStateControl` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Was not relevant for few versions already.

`formControl` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Was not relevant for few versions already.

`inline` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Textarea is now `inline-block` by default.
You can add some style directly on the component if you want to override that.

`readonly` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `isReadonly` attribute to obtain the same behavior.

`required` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `isRequired` attribute to obtain the same behavior.

`resizable` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `isResizable` attribute to obtain the same behavior.

`size ` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

Form components does not provide size variant anymore.
You can use custom CSS if you need to customize the component.

`spellcheck` <OdsBadge color="information" label="Updated" size="sm" />

Has been updated.

You can use the new `hasSpellcheck` attribute to obtain the same behavior.

`textAreaId ` <OdsBadge color="critical" label="Removed" size="sm" />

Has been removed.

This was used to manage a default aria value which is not set by default.
