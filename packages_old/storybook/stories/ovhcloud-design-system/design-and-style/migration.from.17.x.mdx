<Heading label="Style - migrate from v17 to v18" level={ 1 } />

<Heading label="Component tokens" level={ 2 } />

* <OdsBadge color="critical" label="Removed" size="sm" /> All **component-specific tokens** (ex: **`--ods-size-button-*`**, **`--ods-toggle-*`**) have been removed, their values are now directly applied on components.

<Heading label="Color tokens" level={ 2 } />

* <OdsBadge color="critical" label="Removed" size="sm" /> All **primitive color tokens** (ex: **`--ods-color-blue-500`**) have been removed, now colors can only be accessible using alias tokens (see corresponding table below).
* <OdsBadge color="critical" label="Removed" size="sm" /> All **contrasted primitive color tokens** (ex: **`--ods-color-blue-500-contrasted`**) have been removed, until new designs and specifications gets validated for having contrasted states on components.
* <OdsBadge color="critical" label="Removed" size="sm" /> All **`--ods-color-*-list`** tokens have been removed, as no usage for them in integration.

<Heading label="Size tokens" level={ 2 } />

* <OdsBadge color="critical" label="Removed" size="sm" /> All **global size tokens** (ex: **`--ods-size-*`**) have been removed due to no direct use, they are now up to your particular usage.
* <OdsBadge color="critical" label="Removed" size="sm" /> Global **sizing ratio** (ex: **`--ods-size-ratio`**) which was applied to every spacing concept has been removed due to no use.
* <OdsBadge color="critical" label="Removed" size="sm" /> All **spacing concept tokens** (ex: **`--ods-size-stack-*`**) has been removed due to no use.
